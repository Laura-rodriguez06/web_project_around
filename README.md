# Proyecto: Around The U.S.

Este proyecto es una página interactiva donde los usuarios pueden editar su perfil, agregar tarjetas con imágenes y verlas en tamaño completo.

## Funcionalidades principales

- Editar nombre y descripción del perfil.
- Añadir nuevas tarjetas.
- Dar "like" con un sistema de 3 estados (inactivo, hover y activo).
- Ver imágenes en un popup grande.
- Diseño completamente responsive.
- Uso de metodologías BEM.

## Tecnologías utilizadas

- HTML
- CSS (BEM)
- JavaScript
- Normalize.css

## Estructura del proyecto

El proyecto está organizado con carpetas según la metodología BEM:

blocks/
pages/
vendor/
scripts/
images/
